# Laravalidation

Test [Laravel](https://laravel.com) validation rules with ease at [laravalidation.com](https://laravalidation.com).

Started out as a lab project for learning [Vue.js](https://vuejs.org).

## Quickstart

```shell
cp .env.example .env
touch database/database.sqlite

composer install

npm install
npm run watch

php artisan serve
```
